Title:
------
conch


Partner Information:
--------------------
Robert Andrews (robbiiie)
Michael Cheung (michaelc97)


Description:
------------
This plugin answers a question that the user provides. Different length questions will produce different types of responses.
All answers can be deterministically reproduced.

Usage:
------
"conch <anything>"
esh> conch Is the sky on fire?


Testing:
--------
Add this plugin's test file to the plugin.tst file you created. Then run the driver
