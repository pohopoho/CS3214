Title:
------
toBin


Partner Information:
--------------------
Robert Andrews (robbiiie)
Michael Cheung (michaelc97)


Description:
------------
This plugin prints out the binary representation of a number between 0 and 15. A number can 
be converted into binary when the plugin in involked as "toBin <#>". It also handels errors 
involving the case when the user does not type in any number after "toBin", when the user
enters a number less than 0 or greater than 15, and ensuring the plugin name was typed correctly


Usage:
------
"toBin <#>"
esh> toBin 15


Testing:
--------
Add this plugin's test file to the plugin.tst file you created. Then run the driver
