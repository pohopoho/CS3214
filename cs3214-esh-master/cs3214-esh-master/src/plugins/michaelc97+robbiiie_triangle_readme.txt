Title:
------
triangle


Partner Information:
--------------------
Robert Andrews (robbiiie)
Michael Cheung (michaelc97)


Description:
------------
This plugin finds the area of a triangle when supplied the base and height vales of the 
triangle. It accepts the height of the triangle as the first parameter and the base of the
triangle as the second parameter. It handels errors involving the correct name of the plugin
was types, numbers were typed in for the base and height values and not NULL, and the entered 
values are positive ints.


Usage:
------
"triangle <#> <#>"
esh> triangle 10 5


Testing:
--------
Add this plugin's test file to the plugin.tst file you created. Then run the driver
